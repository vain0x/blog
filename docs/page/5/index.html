1. 要約

  • 書き出し
    • ベンチマークは意外と大変という話について軽く触れる
  • 導入
    • 本稿の構成
    • 開発環境の明記
    • BenchmarkDotNet のサンプル
  • 例
    • 一例として「素数判定」を実装する
    • 素数判定の簡単なベンチマークを書く
    • ベンチマークを実行する
  • おまけ
    • 軽くて速いベンチマークに設定する
    • ベンチマークを watch する
